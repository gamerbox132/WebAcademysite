# web

## Project Overview
- Template: Maven Web App
- Frontend: JSP
- Java version: java version "21.0.8" 2025-07-15 LTS
- Tomcat version: 10.1.55.0
- Node version: v24.11.0
- NPM version: 11.6.1

## Quick Setup
1. Build the project with Maven/Gradle
2. Run the app with mvn spring-boot:run or Tomcat workflow
3. Open browser at http://localhost:8080

## Project layout
- backend: src/main/java, src/main/resources
- web: src/main/webapp
- frontend: frontend/
